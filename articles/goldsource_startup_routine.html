Introduction

The first piece of code that runs when you start the game is the launcher's code. The launcher as a main executable is responsible for loading the main engine module and calling the engines's Run function throughout the engine's interface IEngineAPI. Than the main code for the game is stored there, inside the engine. Apart from loading the engine, the loader is also responsible for other things such as communicating with registry, setting up some command line parameters for the engine and also some graphics settings such as application window width, height, the mode and more.
