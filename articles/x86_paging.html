x86 paging

Introduction

Paging is a memory translation process, where logical (also virtual) addresses are translated to linear (physical) addresses done by the MMU (Memory Management Unit). The translation is done on every virtual address when accessed from the memory (RAM). The virtual memory is divided into equal-size blocks called pages. Each of these blocks has different access rights, for example read-only, read-write, etc.

MMU (Memory management unit)

It is a piece of hardware that is responsible for translating virtual addresses into physical addresses. At the same time it handles memory protection of accessed address.

On modern architectures the MMU is packed within the processor internally. When processor want's to access an address, the address first goes through MMU and then through the local bus to the RAM where it locate the data.
